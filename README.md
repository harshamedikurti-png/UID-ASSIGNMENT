# Android Clone — Google Landing Page Frontend

A modern and responsive frontend recreation of the Android by Google official website. This project is developed as part of the **User Interface Design (UID)** subject to demonstrate frontend development skills using HTML, CSS, and JavaScript.

---

## ✨ Live Demo



.
>
> > 📱 Responsive design works on **Mobile, Tablet, and Desktop.**
> 🔗 **Social media buttons** open Android official pages
> THE DEMO LINK : https://heroic-blini-686005.netlify.app/



---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Responsive Web Design
- Font Awesome 6.6.0 (CDN)

## 🚀 Features

- Fully responsive Android website design
- Full-screen video hero section with CTA buttons
- Gemini AI feature cards (Gemini Live & Circle to Search)
- Phone product cards with hover effects
- Powerful Features section (Private Compute Core, Quick Share, Adaptive Battery)
- Android XR teaser section
- Contact form with demo submission handler
- Social media icon buttons
- Smooth scroll animations and transitions
- Modern dark UI design
- Organized code structure
- Fixed navigation bar and footer section

## 📥 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/harshamedikurti-png/UID-ASSIGNMENT/tree/main/assignment
```

### 2. Open Project Folder

```bash
cd Android-Clone
```

### 3. Run the Website

Simply open `a1.html` in your browser.

**OR**

Use **VS Code Live Server** extension for better experience.

> **Note:** Place `android.mp4` in the same folder as `a1.html` for the hero video to work. Without it, the hero section displays a plain dark background.

## 🎨 UI Design Concepts Used

- Dark theme design (`#0a0a0a` background)
- Google green accent (`#34a853`) throughout
- Modern card layout with rounded corners
- Hover animation effects on cards and buttons
- Responsive grid system using CSS Grid
- Full-screen video hero section
- Fixed blurred navigation bar (`backdrop-filter: blur`)
- User-friendly smooth-scroll navigation
